# Página sobre SO con links

Proyecto web estático con información y enlaces relacionados con **Sistemas Operativos (SO)**.  
Hecho para presentar contenido de forma simple, rápida y accesible desde el navegador.

---

## Descripción del proyecto

Este repositorio contiene una página web informativa sobre **Sistemas Operativos**, incluyendo **links** a recursos externos.
El objetivo es centralizar referencias útiles en un solo lugar con un diseño limpio.

---

## Características

- Página web estática (no requiere backend).
- Sección(es) de información sobre Sistemas Operativos.
- Lista de enlaces a recursos externos.
- Diseño con estilos CSS.

---

## Tecnologías / Lenguajes utilizados

Según la composición del repositorio:

- **HTML** — 67.8%
- **CSS** — 32.2%

## Cómo ejecutar el proyecto (local)

1. Descarga o clona el repositorio:
   ```bash
   git clone https://github.com/Itzel921/P-gina-sobre-SO-con-links.git
   ```
2. Abre el archivo `index.html` en tu navegador:
   - Doble clic, o
   - Click derecho → **Open with / Abrir con** tu navegador

> Si usas VS Code, también puedes usar la extensión **Live Server** para verlo con recarga automática.
